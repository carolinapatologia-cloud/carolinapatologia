# 🐰 👋 Olá, eu sou Alice (Atrás do Coelho)

### Transformo linhas de código em melodia e ideias em software.

Atuo conectando **tecnologia, música, arte e desenvolvimento** para criar experiências digitais únicas, compor histórias e dar ritmo à inovação. 

Sou programadora, cantora e compositora. Minha jornada une a lógica do desenvolvimento de software com a sensibilidade da criação musical, sempre buscando a harmonia perfeita entre o código bem escrito e a arte que toca as pessoas.
